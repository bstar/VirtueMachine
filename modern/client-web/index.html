Controls

`U` Use target mode; arrows move target; `Enter`/`U` confirm; `Esc` cancel

Legacy verb keys: `A C T L G D M U R B I`

`0-9` legacy party/solo keys (mapped; party command flow pending)

`Space` / `Esc` pass turn

Use header Login button (or `Shift+I`) to login/logout

Password recovery requires verified recovery email

Passwords are intentionally insecure in this prototype; never reuse important passwords
